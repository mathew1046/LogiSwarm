# LogiSwarm

Geo-aware swarm intelligence for proactive supply chain disruption detection and rerouting.

## Monorepo Structure
- `backend/` — FastAPI backend service
- `frontend/` — Vue 3 + Vite dashboard

## Quick Start
1. Install backend dependencies: `npm run setup:backend`
2. Install all dependencies: `npm run setup:all`
3. Start frontend dev server: `npm run dev`
